乖离指标 (bias) bias = (1+Z) x (P/HM10)
Z = (P-MA200)/(2xstd200)
HM10 = 10年滚动几何均值
评分用5年滚动窗口百分位，bias越低得分越高。
-
-
MA200 偏离 当前价格 vs 200日均线的偏离百分比 (P-MA200)/MA200。5年滚动窗口内百分位越低（负偏离越大）= 越适合买入。
-
-
10年均值 (HM10) 过去10年（2520个交易日）收盘价的几何均值，代表长期趋势中枢。
-
长期趋势参考
RSI (14) RSI 相对强弱指数（14日）。唯一使用绝对阈值的子评分：<30 超卖（100分），>70 超买（0分）。权重20%。
-
-
波动率 (20日年化) 过去20个交易日的年化波动率。5年滚动窗口内百分位越高 = 市场越恐慌 = 抄底机会（权重15%）。
-
越高 = 越恐慌
52周回撤 当前价格 vs 过去52周最高价的回撤幅度。5年滚动窗口内百分位越低（回撤越深）= 抄底良机（权重15%）。
-
-
综合评分 综合评分 = 0.3x乖离评分 + 0.2xRSI + 0.2xMA200偏离 + 0.15x回撤 + 0.15x波动率
除RSI使用绝对阈值(30/70)外，其余子评分均为 5年滚动窗口百分位（0-100），
高分 = 市场恐慌 = 好买点。
综合评分 <= 观望线（50%分位）→ 市场平稳，观望
观望线 < 综合评分 < 抄底线（85%分位）→ 正常定投
综合评分 >= 抄底线 → 市场恐慌，抄底
绝对上限 >=75 必抄底，绝对下限 <=25 必观望
抄底附加条件：当日价格必须 < 当年开盘价x90%，否则属定投区间
